## description
The Tazmin is a website with the model of ponisha site is that it establishes the relationship between the employer and the freelancer and makes it possible for both parties to reach their project goal by interacting with each other.

includes several pages,eg:product page,single product page,
category,sorting,filter products,blogs,wishlist,JWT authentication,login page,auth with otp,userpanel,
admin panel and more...


Using


- React
- javaScript
- TailwindCss
- MongoDB
- React-query
- JWT

## Features
- Allows freelancers to send request for employer projects.
- Allows freelancers to search between several categories based on their expertise
- Allows employer to create thier projects  and manage them in thier profile
- Allows the employer to choose between the best offer of other freelancers
- Allows registered users to login with otp 



Admin privileges

- Allows block ,confirm user accounts
- allow swich between defferent roles

## Installation

1.Clone or download the repository by running the following command in your terminal:

```bash
git clone https://github.com/Alireza-Naghavian/Tazmin.git
```
2.Install the project dependencies using either npm or yarn:

```bash
npm install
```
or
```bash
yarn
```

5.Create a new database in MongoDB Compass and create the required collections with the same names as the downloaded JSON files. Then, import the data from the downloaded files into the corresponding collections.
